# Product

## Register

product

## Users

Crypto-native individuals, digital nomads, and global citizens who manage multi-currency portfolios (fiat + crypto). They use this app in focused work sessions, often in low-light environments, tracking expenses and net worth across multiple wallets and currencies. They value privacy, efficiency, and data clarity over visual noise.

## Product Purpose

Orbital Finance provides a unified view of global finances, normalizing multiple currencies and crypto assets into a single net worth calculation. It replaces spreadsheets and fragmented finance apps with a privacy-first, locally-stored solution that treats finances as a global portfolio.

## Brand Personality

**Precision, Clarity, Confidence** - Professional tool aesthetic inspired by Linear, Raycast, and modern productivity software. The interface should feel like a precision instrument: calm, focused, and expertly crafted. No playful elements, no decorative excess. Every pixel serves the data.

## Anti-references

- Neon-on-black "hacker" aesthetics
- Glassmorphism as default treatment
- Generic crypto app templates with gradient accents
- SaaS dashboard clichés (hero metrics with big numbers, identical card grids)
- Over-animated interfaces with bounce/elastic easing
- Dark mode "because it looks cool" without functional justification

## Design Principles

1. **Data-first, decoration-last** - Visual hierarchy serves information architecture, not aesthetics
2. **Quiet confidence** - The interface should feel like a professional tool, not a consumer app
3. **Progressive disclosure** - Show what's needed now, reveal complexity on demand
4. **Consistent interaction model** - Same patterns, same responses, no surprises
5. **Privacy as design principle** - Local-first architecture reflected in minimal data collection UI

## Accessibility & Inclusion

- WCAG AA compliance minimum
- Support reduced motion preferences
- High contrast ratios for financial data readability
- Keyboard navigation for all interactive elements
- Clear visual states for all interactive elements
